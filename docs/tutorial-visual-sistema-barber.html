Manual do cliente

Como usar o painel

Este guia ensina o uso completo do painel: clientes, serviços, agenda, pagamentos, histórico, personalização, suporte e backup.

Entrar Clientes Serviços Agenda Pagamentos Histórico Configurações Suporte

1. Entrar no Sistema

Abra o aplicativo

No Android, toque no aplicativo “Sistema Barber”. No computador, execute o arquivo “server.exe” e abra “http://localhost:4000” no navegador.

Informe o acesso

Digite o acesso e a senha enviados pelo suporte. Caso ainda não tenha recebido esses dados, solicite-os antes de iniciar o uso.

Confira a tela inicial

Depois de entrar, use o menu para navegar entre “Clientes”, “Serviços”, “Agenda”, “Histórico” e “Configurações”.

O acesso será enviado pelo suporte.

Guarde seu login e sua senha em local seguro. Não compartilhe esses dados com pessoas que não devem acessar o painel.

Se o sistema mostrar um aviso de licença, assinatura ou bloqueio, entre em contato com o suporte que forneceu o sistema.

2. Clientes

Novo cliente

Clique em “Novo Cliente” e informe nome, telefone e observações importantes.

Editar cliente

Use “Editar” quando o telefone mudar ou quando quiser adicionar alguma observação ao cadastro.

Excluir cliente

Use “Excluir” somente quando tiver certeza. Clientes removidos podem afetar consultas futuras.

Telefone para contato.Recomendado
Observações, preferências ou alergias.Opcional

3. Serviços e Preços

Cadastrar serviço

Abra a área “Serviços” e cadastre cortes, barba, sobrancelha, pigmentação, pacote completo ou qualquer outro atendimento oferecido pelo estabelecimento.

Informar preço

Coloque o valor correto. Esse preço entra automaticamente no total do agendamento.

Atualizar tabela

Quando o estabelecimento alterar os preços, edite o serviço antes de criar novos agendamentos.

Dica: mantenha nomes simples, por exemplo: “Corte”, “Barba”, “Corte + Barba” e “Sobrancelha”.

4. Agendamentos

Criar agendamento

Escolha o cliente, a data, os serviços e confirme o valor total.

Evitar duplicidade

Se o cliente já tiver um agendamento aberto, o sistema pode avisar antes de cadastrar outro.

Editar horário

Se o cliente remarcar, edite o agendamento existente em vez de criar outro desnecessário.

5. Pagamentos

Escolha a forma de pagamento

Informe se o cliente pagou em dinheiro, Pix, cartão ou outra forma aceita pelo estabelecimento.

Marque o status

Use “A pagar” quando ainda não recebeu e “Já pago” quando o valor já tiver sido recebido.

Anexe o comprovante, se necessário

Quando houver Pix ou outro comprovante, use o campo de anexo para manter o registro no atendimento.

6. Finalizar Atendimento

Conferir

Antes de finalizar, confira cliente, serviços, valor, data e pagamento.

Finalizar

Ao finalizar, o atendimento sai da agenda ativa e vai para o “Histórico”.

Cancelar ou excluir

Use essa opção apenas quando o atendimento não deve ficar registrado como concluído.

7. Histórico

Consultar atendimentos antigos

Use o histórico para verificar quais serviços foram realizados para cada cliente.

Conferir recebimentos

Veja valores, formas de pagamento e status dos atendimentos finalizados.

Acompanhar frequência

O histórico ajuda a lembrar quando o cliente foi atendido pela última vez.

8. Configurações do estabelecimento

Nome do estabelecimento

Atualize o nome exibido no sistema sempre que necessário.

Cores do painel

Ajuste a cor principal e a borda para combinar com a identidade visual do estabelecimento.

Comprovante Pix

Se desejar, exija comprovante antes de finalizar atendimentos pagos via Pix.

Evite alterar configurações durante horários de grande movimento. Faça ajustes em momentos mais tranquilos.

9. Suporte, Bugs e Recomendações

Enviar recomendação

Use essa opção quando tiver uma sugestão de melhoria, uma nova função ou um ajuste visual.

Reportar bug

Descreva o problema com clareza. Se possível, anexe uma imagem ou um comprovante do erro.

Aguardar retorno

O suporte analisará o relato e informará quando houver correção, orientação ou atualização disponível.

10. Backup e Cuidados

No Windows

Com o sistema fechado, copie o arquivo “database.sqlite” da pasta “Windows”.

No Android

Evite desinstalar o aplicativo sem antes falar com o suporte, pois dados locais podem ser apagados.

Atualizações

Quando receber uma nova versão, siga a orientação do suporte para instalar corretamente.

Importante: não apague os arquivos da pasta “Windows”. O executável precisa da pasta “frontend”, do banco de dados e do arquivo “better_sqlite3.node”.
