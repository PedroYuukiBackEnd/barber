Tutorial visual de uso

Sistema Barber

Guia rapido para instalar, entrar no painel e usar as principais funcoes do sistema no dia a dia da barbearia.

Android

Instale o APK release no celular e abra o aplicativo Sistema Barber.

Windows

Abra o executavel local e acesse o sistema pelo navegador no proprio computador.

Licenca

O acesso e validado pela internet conforme o plano cadastrado pelo administrador.

1. Primeiro Acesso

Abra o sistema

No Android, toque no app instalado. No Windows, execute o sistema local e abra o endereco indicado.

Digite o login inicial

Use as credenciais entregues junto com o sistema. Depois, troque a senha se desejar.

Confira a tela principal

Ao entrar, voce vera as areas de clientes, servicos, agenda, historico e configuracoes.

Acesso inicial: pedroyuuki2008
Senha inicial: pedroyuuki2008

2. Cadastrar Clientes

Nome do clienteObrigatorio
TelefoneOpcional
ObservacoesOpcional

Use a area Clientes para salvar nome, telefone e observacoes. Esses dados ajudam a agendar mais rapido e manter historico.

3. Cadastrar Servicos

Abra Servicos

Cadastre cortes, barba, sobrancelha, pigmentacao ou qualquer outro servico da barbearia.

Informe preco

O valor cadastrado sera usado automaticamente na hora de criar o agendamento.

Edite quando precisar

Se o preco mudar, edite o servico para manter os proximos agendamentos corretos.

4. Criar Agendamentos

Escolha o cliente

Selecione um cliente ja cadastrado.

Escolha os servicos

Marque um ou mais servicos para compor o atendimento.

Defina data e pagamento

Informe data, forma de pagamento e observacoes se necessario.

5. Finalizar Atendimento

Confira o agendamento

Antes de finalizar, veja cliente, servicos e valor total.

Marque como finalizado

O atendimento sai da agenda ativa e vai para o historico.

Consulte depois

Use o historico para ver atendimentos anteriores e valores recebidos.

6. Painel Admin Master

Cadastrar barbearia

Cria o acesso do cliente e gera a licenca remota.

Assinatura

Clientes mensais bloqueiam se vencerem ou forem marcados como vencidos.

Pagamento completo

Clientes desse tipo nao bloqueiam por data, mas ainda podem ser bloqueados manualmente.

7. Suporte e Backup

Backup no Windows

Copie o arquivo database.sqlite com o sistema fechado.

Atualizacao

Receba uma nova versao do APK ou da pasta Windows e instale por cima quando indicado.

Licenca

Se o sistema bloquear, entre em contato com o suporte para regularizar o acesso.
