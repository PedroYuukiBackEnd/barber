Manual do cliente

Como usar o painel da barbearia

Este guia ensina o uso completo do painel que fica na barbearia: clientes, servicos, agenda, pagamentos, historico, personalizacao e suporte.

Entrar Clientes Servicos Agenda Pagamentos Historico Configuracoes Suporte

1. Entrar no Sistema

Abra o aplicativo

No Android, toque no app Sistema Barber. No computador, execute o arquivo server.exe e abra http://localhost:4000.

Digite o acesso

Use o login e a senha iniciais abaixo. O acesso deve ser digitado exatamente como aparece.

Confira a tela inicial

Depois de entrar, use o menu para navegar entre Clientes, Servicos, Agenda, Historico e Configuracoes.

Acesso inicial: pedroyuuki2008
Senha inicial: pedroyuuki2008

Se o sistema mostrar aviso de licenca ou assinatura, entre em contato com o suporte que forneceu o sistema.

2. Clientes

Novo cliente

Clique em Novo Cliente, informe nome, telefone e observacoes importantes.

Editar cliente

Use Editar quando o telefone mudar ou quando quiser adicionar alguma observacao.

Excluir cliente

Use somente quando tiver certeza. Clientes removidos podem afetar consultas futuras.

Nome do clienteObrigatorio
Telefone para contatoRecomendado
Observacoes, preferencias ou alergiasOpcional

3. Servicos e Precos

Cadastrar servico

Abra a area Servicos e cadastre cortes, barba, sobrancelha, pigmentacao, pacote completo ou qualquer outro atendimento.

Informar preco

Coloque o valor correto. Esse preco entra automaticamente no total do agendamento.

Atualizar tabela

Quando a barbearia mudar os precos, edite o servico antes de criar novos agendamentos.

Dica: mantenha nomes simples, por exemplo: Corte, Barba, Corte + Barba, Sobrancelha.

4. Agendamentos

Criar agendamento

Escolha o cliente, a data, os servicos e confirme o valor total.

Evitar duplicidade

Se o cliente ja tiver agendamento aberto, o sistema pode avisar antes de cadastrar outro.

Editar horario

Se o cliente remarcar, edite o agendamento em vez de criar outro desnecessario.

Cliente selecionadoPasso 1
Servicos escolhidosPasso 2
Data, pagamento e observacoesPasso 3

5. Pagamentos

Escolha a forma de pagamento

Informe se o cliente pagou em dinheiro, Pix, cartao ou outra forma configurada.

Marque o status

Use A Pagar quando ainda nao recebeu e Ja Pago quando o valor ja entrou.

Anexe comprovante se precisar

Quando houver Pix ou comprovante, use o campo de anexo para manter registro.

6. Finalizar Atendimento

Conferir

Antes de finalizar, veja cliente, servicos, valor, data e pagamento.

Finalizar

Ao finalizar, o atendimento sai da agenda ativa e vai para o Historico.

Cancelar ou excluir

Use apenas quando o atendimento nao deve ficar registrado como concluido.

7. Historico

Consultar atendimentos antigos

Use o historico para verificar quais servicos foram feitos para cada cliente.

Conferir recebimentos

Veja valores, formas de pagamento e status dos atendimentos finalizados.

Acompanhar frequencia

O historico ajuda a lembrar quando o cliente cortou pela ultima vez.

8. Configuracoes da Barbearia

Nome da barbearia

Atualize o nome exibido no sistema.

Cores do painel

Ajuste a cor principal e a borda para combinar com a identidade da barbearia.

Comprovante Pix

Se desejar, exija comprovante antes de finalizar atendimentos pagos via Pix.

Evite alterar configuracoes durante atendimento movimentado. Faca ajustes em horarios tranquilos.

9. Suporte, Bugs e Recomendacoes

Enviar recomendacao

Use quando tiver sugestao de melhoria, nova funcao ou ajuste visual.

Reportar bug

Descreva o problema com clareza. Se puder, anexe imagem ou comprovante do erro.

Aguardar retorno

O suporte analisa o relato e informa quando houver correcao ou orientacao.

10. Backup e Cuidados

No Windows

Com o sistema fechado, copie o arquivo database.sqlite da pasta Windows.

No Android

Evite desinstalar o app sem antes falar com o suporte, pois dados locais podem ser apagados.

Atualizacoes

Quando receber nova versao, siga a orientacao do suporte para instalar corretamente.

Importante: nao apague os arquivos da pasta Windows. O executavel precisa da pasta frontend, do banco e do arquivo better_sqlite3.node.
